I am Hacker
====
## Overview
ハッカーになった気分になれるジョークツールです。  
デフォルトで出力されるコードは一応動作します。

## Description
キーボードの任意のキーを押すことで、画面にテキストを出力できます。

## Requirement
- Go 1.10 or later

## Install
```bash:#
go get github.com/yuki9431/I_am_hacker

go build
```

## How to Use
```go:main.go
$ ihacker 

or

$ ihacker -t text.c
```
